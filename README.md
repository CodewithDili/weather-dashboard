# Weather Dashboard

A simple Python script to fetch and display current weather data using the OpenWeatherMap API.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/CodewithDili/weather-dashboard.git
   cd weather-dashboard
Install the required libraries:
pip install requests

Sign up for an API key from OpenWeatherMap and replace your_openweathermap_api_key in weather_dashboard.py with your actual API key.

To run the script, use the following command:
python weather_dashboard.py

When prompted, enter the name of the city for which you want to get the weather information.

Features
Fetch current weather data for a specified location.
Display temperature, humidity, wind speed, and weather conditions.
